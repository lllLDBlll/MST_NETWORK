# MST_NETWORK
Minimum spanning tree as computer network topology example.

## Features:
#### - MST implementation (Prim's)
#### - Receive .txt graph (or .csv)
#### - Output graph .dot (as .txt)
#### - Datas by IP/MAC/GATEWAY in vertexes
